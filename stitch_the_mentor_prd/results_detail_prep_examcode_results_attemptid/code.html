Question 12 Trigonometry • Difficulty: Hard

If sin(x) = 0.6 and x is an acute angle, what is the value of cos(2x)?

0.36 Your Answer
check
0.28 Correct Answer
0.64
0.8
lightbulb Explanation

Use the double angle identity for cosine: cos(2x) = 1 - 2sin²(x).
Substitute sin(x) = 0.6:
cos(2x) = 1 - 2(0.6)² = 1 - 2(0.36) = 1 - 0.72 = 0.28.
